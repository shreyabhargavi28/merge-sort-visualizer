# 🔄 Merge Sort Visualizer with Puzzle-Based Learning

An interactive, educational web application that visualizes the Merge Sort algorithm through an engaging puzzle-based interface.

## ✨ Features

- **Dual-Panel Design**: Algorithm visualization on left, puzzle progress on right
- **Step-by-Step Navigation**: Previous/Next buttons to walk through each merge operation
- **Adjustable Speed**: Control animation speed (100ms - 1000ms)
- **Custom Arrays**: Input your own values to sort
- **Multiple Image Modes**: 
  - 🏔️ Nature Mode (mountain landscape)
  - 🎨 Gradient Mode (colorful gradient)
  - 🧠 Learning Mode (brain & light bulb theme)
- **Code Display**: View Merge Sort implementation in 4 languages (Python, JavaScript, Java, C++)
- **Puzzle Progress**: Image reveals itself as sorting completes
- **Keyboard Shortcuts**: Arrow keys, Spacebar, 'R' key
